## Running Examples Locally

```bash
# Clone the repo
git clone https://github.com/linq2js/storion.git
cd storion

# Install dependencies
pnpm install

# Run feature showcase
pnpm --filter feature-showcase dev

# Run Pokemon app
pnpm --filter pokemon dev
```
